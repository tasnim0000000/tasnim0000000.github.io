# Tasnim Ahmed — Portfolio

Personal portfolio website for Tasnim Ahmed — Frontend Developer & UI/UX Designer.
Built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies, no build step.

## 📁 Folder Structure

```
portfolio/
├── index.html          ← Main entry point
├── css/
│   └── style.css       ← All styles (design tokens, layout, animations)
├── js/
│   └── main.js         ← Cursor, scroll reveals, theme toggle, animations
├── assets/             ← Images, icons, resume PDF (add yours here)
└── README.md
```

## 🚀 Hosting on GitHub Pages

1. Push this repo to GitHub as `tasnim0000000.github.io`
2. Go to **Settings → Pages**
3. Set Source to: **Deploy from branch → main → / (root)**
4. Your site will be live at: `https://tasnim0000000.github.io`

## ✏️ Customizing

- **Projects**: When ready, replace the "Coming Soon" block in `index.html` (`#projects` section) with project cards
- **Photo**: Replace the initials placeholder in the About section with an `<img>` tag pointing to `assets/photo.jpg`
- **Resume**: Add your resume PDF to `assets/` and link the "Hire Me" button to it
- **Colors**: All design tokens are CSS variables at the top of `css/style.css`
